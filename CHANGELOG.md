- **Complete overhaul of the Ice Maze generation** (backported from the upcoming Aquamirae 7)!
Instead of dense clusters of wall-like structures that didn't truly form a proper maze, the generation now uses a multi-layered noise-based system. This makes exploration more immersive and visually diverse, while also significantly improving performance by reducing the number of generated structures several times over.
- **Weather inside the Ice Maze has been adjusted** – it now properly snows within the biome.
This is a temporary implementation and not as robust as the system planned for Aquamirae 7, so in large modpacks with custom world generation it may not always apply correctly.
- **Added a subtle ambient background loop** to reinforce the atmosphere of the frozen wasteland.
- **Removed procedurally generated loot armor from the Creative tab**.
It can still be obtained as drops, but will no longer clutter your JEI listings.
